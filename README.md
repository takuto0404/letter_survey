# 文字の特徴に関する調査

このリポジトリの `index.html` が GitHub Pages にデプロイされる想定です。ここで行った修正はローカルの Git リポジトリに保存されますが、GitHub のリモートに `git push` するまでは公開ページに反映されません。

## GitHub Pages に反映する手順
1. 変更内容をコミットする（この環境では既にコミット済みになります）。
2. GitHub 上の公開用ブランチ（例: `main` や `gh-pages`）にプッシュする。
3. GitHub Pages の設定で参照ブランチが合っていることを確認する。

`git status` でローカルの変更状況を確認でき、`git log` でコミット履歴を確認できます。GitHub Pages への反映はプッシュ後に数十秒〜数分かかる場合があります。
